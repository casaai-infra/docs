## Welcome to CasaAI

CasaAI is an AI-driven property management automation platform for real estate professionals that uses AI agents to automate leasing, tenant management, financial reporting, maintenance automation, document generation, and portfolio insights. This documentation helps you understand the product, set up quickly, and scale your workflows.

## What this documentation covers

- Product overview and value: how CasaAI reduces manual work, speeds workflows, and centralizes data.
- Guided onboarding: sign up, connect data, configure properties, train agents, and run sample workflows.
- Feature deep dives: tenant onboarding, rent collection, maintenance automation, reporting, documents, dashboards.
- Developer reference: API/agents, webhooks, SDK snippets, data ingestion patterns.
- Tutorials, use cases, FAQs, troubleshooting, integrations, and ROI measurement.
